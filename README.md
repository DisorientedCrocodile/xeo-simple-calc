# xeo-simple-calc
This is my first python library. It is a simple calculator.

## How to use
First, install the library
```python
pip install xeo-simple-calc
```
Next, import it
```python
from xeosimplecalc import *
```
## Commands
```python
add(a,b)
subtract(a,b)
multiply(a,b)
float_divide(a,b)
int_divide(a,b)
```
```python
float_divide()
``` 
returns a float
```python
int_divide()
``` 
returns an int
